# Nuxt 3  - Desafio Mlabs

Link para acessar o projeto: https://desafio-mlabs.vercel.app/

## O que foi utilizado: 

- Nuxt.js 3
- Typescript
- Pinia 

## Setup

- Instalar as dependencias do projeto

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Server

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
