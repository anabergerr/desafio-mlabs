# 🚀 Nuxt 3 - Desafio Mlabs

🔗 **Link para acessar o projeto**: [desafio-mlabs.vercel.app](https://desafio-mlabs.vercel.app/)

## 🛠️ O que foi utilizado:

- Nuxt.js 3
- TypeScript
- Pinia

## 📦 Setup

1. Instale as dependências do projeto:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## 🖥️ Server

2. Inicie o servidor de desenvolvimento:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
